# SONGVERSE: 中国音乐出海服务体系标准
# SONGVERSE: China Music Global Service System Standard

**版本 / Version:** Draft v0.1  
**状态 / Status:** 开源实践版 / Open-source Practice Version  
**发布源 / Canonical Source:** 本 GitHub 仓库为本标准的权威发布源  
**许可 / License:** 待定（建议未来添加 LICENSE 文件）

SONGVERSE 是一个面向中国音乐内容“全球发行、跨文化推广、数据与流程规范”的开放式行业参考框架。  
SONGVERSE 作为一个文档体系名称使用，不代表商标、组织或任何法律主体。

SONGVERSE is an open, voluntary documentation framework for global distribution and cross‑cultural promotion of music originating from China.  
The name “SONGVERSE” refers only to the documentation framework and does not represent a trademark, organization, or legal entity.

SONGVERSE 为中国音乐人、创作者、团队及相关从业者提供一套可操作、可复制、可审计的音乐出海流程，包括：

**English:**

## 4. 标准流程 / Standard Process

SONGVERSE 标准遵循欧美音乐行业普遍采用的“生命周期式工作流”（Lifecycle Workflow），  
覆盖从前期准备到上线后的增长与结算。每一阶段均包含可执行步骤和可验证产出（deliverables）。

The SONGVERSE Standard follows a lifecycle workflow commonly used in EU/NA music industries,  
covering all steps from pre-release preparation to long-term growth and revenue reconciliation.

---

### **PREPARE — 全球发行准备 / Global Release Preparation**

**中文：**
- 明确主版本与附加版本（Radio Edit、Instrumental 等）
- 准备全球统一元数据（ISRC、UPC、Composer、Lyricist 等）
- 准备国际通用视觉封面（3000×3000）
- 整理 DSP 素材（Audio、Visual、Press Kit）
- 制定目标市场与目标受众画像
- 创建发行节奏与预热计划（Release Plan）

**English:**
- Define master track and versions (Radio Edit, Instrumental, etc.)
- Prepare global metadata (ISRC, UPC, composers, lyricists)
- Create DSP‑compliant cover art (3000×3000)
- Prepare DSP asset package (audio, images, press materials)
- Define target markets and audience profiles
- Build the pre-release plan and timeline

**Deliverables:**  
Metadata Package / Asset Package / Release Plan / Targeting Brief

---

### **DISTRIBUTE — 全球分发 / Global Digital Distribution**

**中文：**
- 按发行商标准交付 WAV + Cover + Metadata
- 向全球 DSP 提交内容
- 设置 Pre-save / Pre-add（欧美常规）
- 确保发行日跨时区一致策略
- 处理 DSP 审核反馈
- 追踪平台状态（Processing → Live）

**English:**
- Deliver WAV, cover art, metadata in standard formats
- Distribute to global DSPs via distributor
- Set up pre-save / pre-add campaigns
- Manage time-zone aligned release timing
- Resolve QC issues from DSPs
- Track ingestion progress

**Deliverables:**  
DSP Delivery Confirmation / Pre-save Link / QC Checklist

---

### **PROMOTE — 国际推广 / Global Promotion (EU/NA Focus)**

推广拆分为欧美行业常用的四条并行轨道（Tracks）：

**1. Editorial Outreach（编辑部 Pitch）**  
- 提交国际格式 Pitch  
- 监控 Editorial Review Queue  
- 撰写作品背后故事（Story Angle）

**2. Audience Marketing（受众增长）**  
- Instagram / TikTok / YouTube 国际内容  
- A/B 测试内容形式（Short-form Clips, Lyric Cuts）  
- 广告小额跑数（Test Budget Strategy）

**3. Influencer / UGC Activation（创作者扩散）**  
- 设计趋势 Hook  
- 激活海外 UGC 创作者参与内容扩散  

**4. Press / Community（软传播）**  
- 撰写英文 Press Release  
- 推荐给博客、播放列表策展人（Curators）  
- 社区讨论场景曝光（Forums, Niche Communities）

**Deliverables:**  
Editorial Pitch / Content Package / UGC Brief / Press Kit

---

### **MONITOR — 数据监测与优化 / Monitoring & Optimization**

**中文：**
- 每日播放、收藏、播放完成度统计  
- 分区域表现对比（Territories）  
- 社媒指标：ER、CTR、Retention  
- 找出 Strong/Weak Assets  
- 调整内容输出节奏与策略

**English:**
- Daily streams, saves, and completion rate tracking  
- Territory performance analysis  
- Social metrics: ER, CTR, retention curves  
- Identify strong vs. weak content  
- Adjust pacing, targeting, and creatives

**Deliverables:**  
Weekly Performance Report / Optimization Plan

---

### **REVENUE — 收益结算 / Revenue Settlement**

**中文：**
- 跟踪 DSP 月度或季度报表  
- 区分地区与收益类型（Streams、UGC、Downloads）  
- 数据与报表对账  
- 出具收益跟踪文件

**English:**
- Track monthly/quarterly DSP statements  
- Segment revenue by territory & type  
- Perform reconciliation  
- Generate revenue summary reports

**Deliverables:**  
Revenue Statement / Reconciliation Sheet

---

### **SERVICE — 长期运营 / Long‑term Operations**

**中文：**
- 制定滚动式发行计划（Release Cadence）
- 持续建设国际粉丝社区（Growth Loop）
- 持续测试与优化内容策略
- 根据市场变化调整定位

**English:**
- Build long-term release cadence  
- Develop global fan & community growth loops  
- Conduct continuous testing  
- Adjust positioning based on market trends

**Deliverables:**  
Release Roadmap / Community Growth Plan
